📥 Импорт достижений из JSON

← Назад к списку

Вставьте JSON с достижениями в поле ниже. Поддерживаются форматы:

  • Объект с ключом "achievements" → массив достижений
  • Массив объектов достижений
  • Одиночный объект достижения

Каждое достижение должно содержать name, и опционально — description, icon, condition_type, condition_value, xp_reward. Если достижение с таким же названием уже существует — оно будет обновлено.
