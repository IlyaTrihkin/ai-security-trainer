📥 Импорт уроков из JSON

← Назад к списку

Вставьте JSON с уроками в поле ниже. Поддерживаются форматы:

  • Объект с ключом "lessons" → массив уроков
  • Массив объектов уроков
  • Одиночный объект урока

Каждый урок должен содержать title, skill_id, и опционально — topics и questions. Если урок с таким же названием и skill_id уже существует — он будет обновлён.
