# 🧠 AI Security Trainer

Интерактивная образовательная платформа по информационной безопасности с геймификацией, адаптивными сценариями и системой достижений.  
**«Duolingo для ИБ»** — учись безопасному поведению в интернете и на работе, зарабатывай XP, получай бейджи и соревнуйся с коллегами.

---

## 🚀 Возможности

- 🧠 **Интерактивные уроки** — пошаговая лекция по блокам + тест с вопросами по одному и мгновенной обратной связью.
- 📝 **Поддержка открытых вопросов** — текстовый ввод ответа с проверкой по ключевым словам (без вариантов).
- 🛡️ **Админ-панель** — управление пользователями, навыками, уроками и достижениями (CRUD), импорт/экспорт в JSON, пагинация и поиск.
- 🔄 **Повторное прохождение** — возможность пройти урок заново, сохранение лучшего результата и подсчёт попыток.
- 🤖 **ИИ-помощник (YandexGPT)** — встроенный чат-бот для вопросов по информационной безопасности, генерация дополнительных тестовых вопросов к урокам.
- 📰 **Раздел новостей** — RSS-агрегация новостей по кибербезопасности (русские и английские источники: SecurityLab, Xakep, The Hacker News, Krebs on Security и др.).
- 📊 **Тепловые карты и прогресс-бары** — визуализация прогресса по урокам и навыкам на дашборде и странице навыков.
- 🏆 **Геймификация** — XP, уровни (каждые 100 XP), достижения (бейджи), лидерборд, стрики.
- 📸 **Аватарки пользователей** — загрузка фото через профиль, безопасная обработка (ограничение размера, проверка типа, уникальные имена).
- 🌳 **Карта навыков** — группировка по уровням сложности (beginner, intermediate, advanced), прогресс-бары и тепловые карты по каждому навыку.
- 📈 **Лидерборд** — рейтинг пользователей по XP с местами, аватарами и количеством пройденных уроков.
- 👤 **Профиль** — просмотр и редактирование данных, смена пароля (с защитой от timing attacks через `hmac.compare_digest`), загрузка/удаление аватара.
- 🌙 **Тёмная тема** — адаптивный дизайн, боковое меню, мобильная адаптация, скруглённые элементы, неоновые акценты.
- 🔒 **Безопасность** — защита от timing attacks (`hmac.compare_digest`), безопасная загрузка файлов (проверка размера, типа, TOCTOU-защита через проверку после сохранения), безопасный random (`secrets`), отключён debug в production.
- 🐳 **Docker** — быстрый запуск через Docker Compose (или локально без Docker).

| Компонент          | Технология                                                     |
|--------------------|----------------------------------------------------------------|

| Markdown            | Python-Markdown (рендеринг контента уроков с подсветкой кода) |

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/IlyaTrihkin/ai-security-trainer.git
cd ai-security-trainer
```

### 2. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example` и заполните необходимые ключи:

```bash
cp .env.example .env
```

Минимально необходимые переменные:

```env
SECRET_KEY=your-secret-key-here
DATABASE_URL=sqlite:///instance/security_trainer.db

# Для YandexGPT (чат-бот и генерация вопросов)
YANDEX_API_KEY=your-yandex-api-key
YANDEX_FOLDER_ID=your-yandex-folder-id
```

### 3. Запуск через Docker (рекомендуемый способ)

```bash
docker compose up -d --build
```

После запуска откройте в браузере: http://localhost:5000

Инициализация базы данных (только при первом запуске):

```bash
docker compose exec backend python init_db.py
```

### 4. Локальный запуск (без Docker, для разработки)

```bash
# Установите зависимости
python3 -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt

# Создайте папку instance и инициализируйте базу данных
mkdir -p instance
python3 backend/init_db.py

# Запустите приложение
python3 run.py
```

Приложение будет доступно по адресу http://127.0.0.1:5000.

## 🛡️ Администрирование

Платформа включает встроенную админ-панель для управления контентом и пользователями.

### Вход в админ-панель

1. Зарегистрируйтесь в приложении как обычный пользователь.
2. Подключитесь к базе данных и назначьте пользователю права администратора:

```sql
-- Для SQLite:
UPDATE user SET is_admin = 1 WHERE username = 'ваш_логин';

-- Для PostgreSQL (через Docker):
docker compose exec db psql -U security_user -d security_db -c "UPDATE \"user\" SET is_admin = true WHERE username = 'ваш_логин';"
```

3. После повторного входа в приложение вы увидите пункт «Админ-панель» в навигационном меню.

### Возможности админ-панели

| Раздел | Описание |
|--------|----------|
| **Пользователи** | Просмотр списка, поиск, пагинация, блокировка/разблокировка, выдача/отзыв прав администратора |
| **Навыки** | Создание, редактирование, удаление навыков (иконки, сложность, родительские навыки) |
| **Уроки** | CRUD-операции, привязка к навыкам, WYSIWYG-редактор (TinyMCE), управление темами и вопросами |
| **Достижения** | Создание/редактирование/удаление бейджей, импорт/экспорт в JSON |
| **Импорт/Экспорт** | Экспорт уроков и достижений в JSON-файл, импорт из JSON с валидацией структуры |

**Адрес админ-панели:**

```
http://localhost:5000/admin/
```

## 🔄 Повторное прохождение уроков

Вы можете пройти любой урок заново — для этого на странице навыков рядом с уже пройденным уроком нажмите кнопку **«🔄 Заново»**. При повторном прохождении:

- Сохраняется **лучший результат** (максимальный счёт) из всех попыток.
- **Счётчик попыток** (`attempts`) увеличивается на 1.
- Новый XP начисляется, только если результат превышает предыдущий.
- Прогресс отображается в формате: `Правильных: X из Y (Z%)`.

## 🧭 Как пользоваться

1. **Зарегистрируйся** — создай аккаунт.
2. **Загрузи аватар** (опционально) — через профиль.
3. **Перейди в карту навыков** — выбери урок по интересующей теме.
4. **Проходи уроки** — изучай лекцию по блокам, затем отвечай на вопросы теста по одному с мгновенной обратной связью. Некоторые вопросы требуют текстового ответа — вводи его в поле и получай оценку сразу.
5. **Используй ИИ-помощник** — задавай вопросы чат-боту (кнопка в правом нижнем углу) по любой теме ИБ.
6. **Генерируй дополнительные вопросы** — на странице урока нажми кнопку «Сгенерировать дополнительные вопросы» для получения новых тестовых заданий через YandexGPT.
7. **Читай новости** — в разделе «Новости» доступны свежие статьи по кибербезопасности из RSS-лент (можно переключать язык: русский/английский).
8. **Следи за прогрессом** — на дашборде отображается тепловая карта всех уроков, прогресс-бары по навыкам, последние пройденные уроки и достижения.
9. **Соревнуйся** — смотри свой рейтинг в лидерборде.
10. **Получай достижения** — бейджи за прогресс.

## 🎯 Новый интерфейс урока

Уроки представлены в формате пошагового обучения:

1. **Лекция (блоки)** — учебный материал разбит на тематические блоки с навигацией «Назад / Вперёд».
2. **Тест (вопросы по одному)** — каждый вопрос показывается отдельно. При выборе варианта сразу отображается обратная связь: ✅ ВЕРНО или ❌ НЕВЕРНО с объяснением и подсветкой правильного ответа.
3. **Открытые вопросы** — если вопрос не имеет вариантов ответа, появляется поле для текстового ввода. Ответ проверяется по ключевым словам (например, критерии надёжного пароля).
4. **Результаты** — итоговый экран с процентом правильных ответов, начисленными XP, информацией о повышении уровня и новых достижениях.
