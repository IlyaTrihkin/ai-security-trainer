# 🧠 AI Security Trainer

Интерактивный тренажёр по информационной безопасности с геймификацией и адаптивными сценариями на основе ИИ.

## 🎯 Цель

Обучить сотрудников основам кибербезопасности через игровые сценарии, диалоги и практические задания. Аналог Duolingo, но для ИБ.

## 📦 Технологии

- Python + Flask
- PostgreSQL
- YandexGPT / GigaChat (ИИ)
- Bootstrap 5
- Docker Compose

## 🚀 Быстрый старт

```bash
git clone https://github.com/IlyaTrihkin/ai-security-trainer.git
cd ai-security-trainer
docker compose up -d
```

## 🚀 Возможности (на данный момент)

- Регистрация и аутентификация пользователей
- Дашборд с профилем и прогрессом
- Модели данных для навыков, уроков, вопросов и достижений
- PostgreSQL для хранения данных
- Docker Compose для быстрого развёртывания
