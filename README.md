# 🧠 AI Security Trainer

Интерактивная образовательная платформа по информационной безопасности с геймификацией, адаптивными сценариями и системой достижений.  
**«Duolingo для ИБ»** — учись безопасному поведению в интернете и на работе, зарабатывай XP, получай бейджи и соревнуйся с коллегами.

---

## 🚀 Возможности

- **🧠 Интерактивные уроки** — теория + практические вопросы по ИБ.
- **🏆 Геймификация** — XP, уровни, достижения (бейджи), лидерборд.
- **📸 Аватарки пользователей** — загрузка фото или инициалы.
- **🌳 Карта навыков** — древовидная структура с прогрессом.
- **📊 Дашборд** — персональная статистика и визуализация прогресса.
- **📱 Адаптивный дизайн** — тёмная тема, боковое меню.
- **🐳 Docker** — быстрый запуск в любой среде.

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/IlyaTrihkin/ai-security-trainer.git
cd ai-security-trainer
```

### 2. Настройка окружения

Скопируй .env.example в .env и укажи свои параметры (если нужно):

```bash
cp .env.example .env
```

### 3. Запуск через Docker

```bash
docker compose up -d --build
```

После запуска открой в браузере: `http://localhost:5000`

### 4. Инициализация базы данных (только при первом запуске)

```bash
docker compose exec backend python init_db.py
```

## 🧭 Как пользоваться

1. Зарегистрируйся — создай аккаунт.

2. Загрузи аватар (опционально) — через профиль.

3. Перейди в карту навыков — выбери урок по интересующей теме.

4. Проходи уроки — отвечай на вопросы, получай XP и уровни.

5. Соревнуйся — смотри свой рейтинг в лидерборде.

6. Получай достижения — бейджи за прогресс.
