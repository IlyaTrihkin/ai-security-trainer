# 🧠 AI Security Trainer

Интерактивная образовательная платформа по информационной безопасности с геймификацией, адаптивными сценариями и системой достижений.  
**«Duolingo для ИБ»** — учись безопасному поведению в интернете и на работе, зарабатывай XP, получай бейджи и соревнуйся с коллегами.

---

## 🚀 Возможности

- 🧠 **Интерактивные уроки** — пошаговая лекция по блокам + тест с вопросами по одному и мгновенной обратной связью.
- 📝 **Поддержка открытых вопросов** — текстовый ввод ответа с проверкой по ключевым словам (без вариантов).
- 🛡️ **Админ-панель** — управление пользователями, навыками, уроками и достижениями (CRUD), импорт/экспорт в JSON.
- 🔄 **Повторное прохождение** — возможность пройти урок заново, сохранение лучшего результата и подсчёт попыток.
- 📄 **Пагинация и поиск** — во всех разделах админ-панели для удобной навигации по большому количеству записей.
- 🏆 **Геймификация** — XP, уровни, достижения (бейджи), лидерборд.
- 📸 **Аватарки пользователей** — загрузка фото через профиль, отображение в навигации и на страницах.
- 🌳 **Карта навыков** — группировка по уровням (от пользователя до эксперта), прогресс по каждому навыку.
- 📊 **Дашборд** — персональная статистика: общий XP, количество пройденных уроков, полученные достижения.
- 📈 **Лидерборд** — рейтинг пользователей по XP с местами и количеством пройденных уроков.
- 👤 **Профиль** — просмотр и редактирование данных, смена пароля, загрузка аватара.
- 📱 **Адаптивный дизайн** — тёмная тема, боковое меню, мобильная адаптация.
- 🔒 **Безопасность** — защита от основных уязвимостей (timing attacks, TOCTOU, debug-режим, безопасный random), аудит через mycop и SigMap.
- 🐳 **Docker** — быстрый запуск в любой среде (или локально без Docker).

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/IlyaTrihkin/ai-security-trainer.git
cd ai-security-trainer
```

### 2. Запуск через Docker (рекомендуемый способ)

Скопируйте `.env.example` в `.env` и при необходимости укажите свои параметры (если не меняете — оставьте как есть).

```bash
cp .env.example .env
docker compose up -d --build
```

После запуска откройте в браузере: http://localhost:5000

Инициализация базы данных (только при первом запуске):

```bash
docker compose exec backend python init_db.py
```

### 3. Локальный запуск (без Docker, для разработки)

Если вы предпочитаете запускать проект без Docker (например, в WSL/Ubuntu):

```bash
# Установите зависимости
python3 -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt

# Настройте переменные окружения в .env (укажите DATABASE_URL для SQLite или PostgreSQL)
# Пример для SQLite:
# DATABASE_URL=sqlite:///instance/security_trainer.db

# Создайте папку instance и инициализируйте базу данных
mkdir -p instance
python3 backend/init_db.py

# Запустите приложение
python3 run.py
```

Приложение будет доступно по адресу http://127.0.0.1:5000.

## 🛡️ Администрирование

Платформа включает встроенную админ-панель для управления контентом и пользователями.

### Вход в админ-панель

1. Зарегистрируйтесь в приложении как обычный пользователь.
2. Подключитесь к базе данных и назначьте пользователю права администратора:

```sql
-- Для SQLite:
UPDATE user SET is_admin = 1 WHERE username = 'ваш_логин';

-- Для PostgreSQL (через Docker):
docker compose exec db psql -U security_user -d security_db -c "UPDATE \"user\" SET is_admin = true WHERE username = 'ваш_логин';"
```

3. После повторного входа в приложение вы увидите пункт «Админ-панель» в навигационном меню.

### Возможности админ-панели

| Раздел | Описание |
|--------|----------|
| **Пользователи** | Просмотр списка, поиск, пагинация, блокировка/разблокировка, выдача/отзыв прав администратора |
| **Навыки** | Создание, редактирование, удаление навыков с иконками и уровнями сложности |
| **Уроки** | CRUD-операции, привязка к навыкам, WYSIWYG-редактор (TinyMCE) для описания и контента, управление темами и вопросами внутри урока |
| **Достижения** | Создание, редактирование, удаление достижений (бейджей), импорт/экспорт в JSON |
| **Импорт/Экспорт** | Экспорт уроков и достижений в JSON-файл, импорт из JSON (с валидацией структуры) |

## 🔄 Повторное прохождение уроков

Вы можете пройти любой урок заново — для этого на странице навыков рядом с уже пройденным уроком нажмите кнопку **«🔄 Заново»**. При повторном прохождении:

- Сохраняется **лучший результат** (максимальный счёт) из всех попыток.
- **Счётчик попыток** (`attempts`) увеличивается на 1.
- Новый XP начисляется, только если результат превышает предыдущий.
- Прогресс отображается в формате: `Правильных: X из Y (Z%)` с ограничением до 100%.

### Адрес админ-панели

```
http://localhost:5000/admin/
```

## 🧭 Как пользоваться

1. **Зарегистрируйся** — создай аккаунт.

2. **Загрузи аватар** (опционально) — через профиль.

3. **Перейди в карту навыков** — выбери урок по интересующей теме.

4. **Проходи уроки** — изучай лекцию по блокам, затем отвечай на вопросы теста по одному с мгновенной обратной связью. Некоторые вопросы требуют текстового ответа — вводи его в поле и получай оценку сразу.

5. **Соревнуйся** — смотри свой рейтинг в лидерборде.

6. **Получай достижения** — бейджи за прогресс.

## 🎯 Новый интерфейс урока

Уроки теперь представлены в формате пошагового обучения:

1. **Лекция (блоки)** — учебный материал разбит на тематические блоки с навигацией «Назад / Вперёд».
2. **Тест (вопросы по одному)** — каждый вопрос показывается отдельно. При выборе варианта сразу отображается обратная связь: ✅ ВЕРНО или ❌ НЕВЕРНО с объяснением и подсветкой правильного ответа.
3. **Открытые вопросы** — если вопрос не имеет вариантов ответа, появляется поле для текстового ввода. Ответ проверяется по ключевым словам (например, критерии надёжного пароля).
4. **Результаты** — итоговый экран с процентом правильных ответов, начисленными XP, информацией о повышении уровня и новых достижениях.

## 🎯 Архитектура проекта

Проект построен по принципу клиент-серверной архитектуры с использованием Docker Compose для оркестрации сервисов.

### Структура

```text
ai-security-trainer/
├── backend/                      # Flask-приложение (Python)
│   ├── app/
│   │   ├── __init__.py           # Фабрика приложения (конфигурация, БД, логин)
│   │   ├── models.py             # SQLAlchemy модели (User, Skill, Lesson, Topic, Question, Progress, Achievement)
│   │   ├── routes.py             # Все маршруты (главная, навыки, уроки, профиль, дашборд, лидерборд, submit, результат)
│   │   ├── admin.py              # Админ-панель (управление пользователями, навыками, уроками, импорт/экспорт JSON)
│   │   ├── decorators.py         # Декоратор @admin_required для защиты маршрутов админ-панели
│   │   └── gamification.py       # Логика проверки достижений
│   ├── data/                     # JSON-контент (уроки, вопросы, скиллы, достижения)
│   │   ├── skills.json
│   │   ├── lessons.json
│   │   ├── lessons_full.json     # Полный дамп уроков для импорта/экспорта
│   │   ├── achievements.json
│   │   └── ...
│   ├── init_db.py                # Скрипт инициализации БД (создание таблиц, загрузка данных)
│   ├── load_lesson.py            # Утилита загрузки уроков из JSON в БД
│   ├── tests/                    # Тесты

│   │       ├── skills.html       # Управление навыками
│   │       ├── skill_form.html   # Форма создания/редактирования навыка
│   │       ├── lessons.html      # Управление уроками
```
